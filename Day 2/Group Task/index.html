Pseudo Elements

What is a Pseudo Element?

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s).

Syntax

selector::pseudo-element {
     property: value;
}

Some Examples

::after

Add content after the content defined in the html document.

::first-letter

Change the styling of the first letter of an element.

::file-selector-button

Change styling of the button that opens the file selection window for file type input element.

::marker

Change the marker infront of list items (The black dot and numbers for unordered and ordered list respectively).

::selection

Change styling of whatever the user has selected with the cursor.
